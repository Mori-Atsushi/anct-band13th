No. 2
Date 2015.7.19

プログラム

第一部

ポップスステージ

第一部は気楽に楽しめるポップス曲を演奏します。吹奏楽オリジナルポップス曲やお馴染みのあの曲で吹奏楽の面白さを感じてください。また、今年も合唱とのコラボを行います。

演奏曲目

  • フライ・ハイ / 星出 尚志
  • ふるさと / 嵐

第二部

パフォーマンスステージ

明石高専史上始めての試みです。普及の名作「映画｜サウンドオブミュージック」を吹奏楽、アンサンブルで完全再現します。懐かしのあの映像を頭に浮かべながらお楽しみください。

演奏曲目

  • ドレミの歌 / R.ロジャース
  • 私のお気に入り / R.ロジャース

第三部

クラシックステージ

第三部では聴き応えのある吹奏楽オリジナル曲を演奏します。明石高専吹奏楽部の生まれ変わったサウンドをお聴きください。

演奏曲目

  • さくらのうた / 福田 洋介
  • マスク / W.F.マクベス

企画① 合唱コラボ

今年で3年目になります。合唱コラボを行います。毎年合唱部とのコラボをということでしたが、今年は全校生徒に声をかけて総勢30名の合唱団と共に演奏します。曲目は大人気ジャニーズグループの嵐がNHK紅白歌合戦で歌っている「ふるさと」です。

企画② アンサンブル

第二部のパフォーマンスステージで行います。映画「サウンド・オブ・ミュージック」のストーリーに沿って、映画中の音楽を吹奏楽の合奏やアンサンブルで演奏していきます。壮大な企画です。ご期待ください。
